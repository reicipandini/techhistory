TECH HISTORY

Software

Software é a parte lógica do computador, ou seja, os programas, sistemas e aplicativos que fazem o computador funcionar e permitem que o usuário realize tarefas.

Sistema Operacional

Responsável por controlar o computador.

O sistema operacional é o software mais importante de um computador. Ele gerencia o hardware, arquivos, memória, programas e permite que o usuário interaja com o computador. Exemplos: Windows, Linux e macOS. Sem o sistema operacional, o computador não funciona. Ele controla teclado, mouse, tela, armazenamento e todos os programas instalados.

Aplicativos

Programas usados pelo usuário.

Aplicativos são programas criados para realizar tarefas específicas, como editar textos, navegar na internet, assistir vídeos, jogar ou programar. Exemplos: Google Chrome, Word, Excel, Photoshop, jogos e editores de vídeo. Existem aplicativos para computadores e celulares.

Software Livre

Programas que podem ser modificados.

Software livre é aquele que qualquer pessoa pode usar, estudar, modificar e distribuir. O código fonte é aberto para desenvolvedores. Exemplos: Linux, LibreOffice, GIMP. Esse tipo de software ajuda no desenvolvimento da tecnologia e educação.

Software Pago

Programas comerciais.

Software pago é desenvolvido por empresas e precisa de licença para uso. Exemplos: Windows, Microsoft Office, Adobe Photoshop. Normalmente possuem suporte técnico, atualizações e mais recursos.

Linguagens de Programação

Usadas para criar softwares.

As linguagens de programação são usadas para criar programas e sistemas. Exemplos: Python, Java, C++, JavaScript. Elas permitem desenvolver sites, aplicativos, jogos, inteligência artificial e sistemas.

Bancos de Dados

Armazenam informações.

Bancos de dados armazenam informações organizadas como usuários, produtos, notas escolares, contas bancárias e sistemas de empresas. Exemplos: MySQL, PostgreSQL, Oracle. São muito importantes para sistemas e sites.
